Расположение RustDesk
==================================
Все папки для удаленных подключений RustDesk находятся по пути:

::

\10.0.3.8\admin_doc\AdresBookRustDesk\RUSTDESK



В данной директории находятся все необходимые папки для подключения через RustDesk.

Подключение через RustDesk
=================================
Подключение выполняется по паролям.

**Для подключения необходимо:**

Открыть папку нужного компьютера в каталоге RustDesk.
Запустить RustDesk.
Указать ID подключения.
Ввести пароль доступа.
Перечень паролей RustDesk

Используемые пароли:

::

3140xyz
3140xyz1
3140Xyz1
3140Xyz1!
