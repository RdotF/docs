==================================
Просмотр камер через Trassir
============================

Данная инструкция описывает процесс настройки серверов и просмотра камер через программу **Trassir**.

# Установка и открытие программы

Установите программу **Trassir** и запустите её.

После открытия программы в верхней части экрана появятся **3 красные полоски** — это меню Trassir.

Нажмите на них для открытия окна программы.

.. image:: images/camera-1.png
:alt: Открытие Trassir
:align: center
:width: 700px

# Настройка серверов

Для просмотра камер необходимо настроить подключение к серверам.

В программе Trassir откройте раздел настройки серверов.

.. image:: images/camera-2.png
:alt: Настройка серверов Trassir
:align: center
:width: 700px

Используются следующие серверы камер:

## Сервер 1

::

IP: 10.0.4.20
Порты: 3080/3081
Login: admin
Password: 3140Xyz!

## Сервер 2

::

# Просмотр камер

Для просмотра камер нажмите кнопку:

::

Монитор

.. image:: images/camera-3.png
:alt: Просмотр камер Trassir
:align: center
:width: 700px

После открытия монитора будут доступны подключенные камеры выбранного сервера.
